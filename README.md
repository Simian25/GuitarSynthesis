# Karplus-Strong String Synthesis

This project is a JUCE-based guitar synthesis plugin that explores physical modeling of plucked strings using the Karplus-Strong algorithm.

## Overview

The plugin uses a simple feedback delay line and filtering structure to create a resonant, guitar-like tone from a short excitation signal. It serves as a lightweight example of digital string synthesis in a real-time audio plugin.

## Features

- JUCE-based audio plugin project
- Karplus-Strong string synthesis
- Real-time audio processing
- Cross-platform plugin structure

## Getting Started

Open the JUCE project file in the repository root and build it with your preferred JUCE/Visual Studio setup.

## Notes

This repository is intended as a starting point for experimenting with physical modeling and guitar-inspired synth sounds.
